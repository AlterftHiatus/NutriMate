jadi gini, aku dengan tim akan bikin projek tentang health tracker berbasis web, alurnya itu, user login/register lalu akan diarahkan ke halaman dashboard, di sini ada beberapa menu yaitu daily, chat bot, score board, dan profil. untuk daily, isinya itu ada nampilin diagram batang dengan parameter 7 hari terakhir dan exp hariannya, ada riwayat/history aktivitas si user, terus ada nampilin exp user sama streak harian, ada menu untuk memulai aktivitas kayak lari, jogging, bersepeda. untuk exp sendiri diperoleh ketika si user melakukan aktivitasnya, setiap aktivitas menghasilkan exp yang berbeda tergantung banyak kalori yang dibakar. untuk streak harian ini akan bertambah 1 setiap harinya ketika si user memulai aktivitasnya di web kami (bukan ketika login), dan akan ke reset jika si user bolos 1 hari saja, untuk halaman chatbot, isinya itu seperti namanya chatbot, yang di khususkan untuk menerima pertanyaan seputar kesehatan. terus untuk halaman score board, isinya itu menampilkan semua user yang menggunakan web kami dan mengurutkannya dari yang tertinggi berdasarkan jumlah exp mereka, terakhir ada profil, menampilkan profil dari user, seperti nama, email, tinggi badan, berat badan.

itu dia sekilas gambaran dari projek yang akan kita bahas, jika ada pertanyaan yang ingin kamu tanyakan, tannyakan saja guna memperakurat jawaban yang dihasilkan
